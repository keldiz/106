Phase 1 -- Using Processing and Serial (or Arduino and Processing with Serial to create a sample hardware interfaces for laptops)

Artist Statement

This is the first time I've used Processing. Together with the Serial Library,
I was able to explore how to communicate with my computer using different functions and commands.
I used Processing to play with Dimmer and Graph. With Dimmer, I can use my mouse to manipulate the power of my LED.
For Graph, I can use my Analog Sensor to control the graph up and down. Learning both of these controls
was very interesting and cool. It made me realize that I can do a lot of things around me using this
controls such as dimming the lights, turning the music up and down and so on.
